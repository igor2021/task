### Создание темы

Для создания скелета темы вполните следующую команду из папки application:

```
$ ./yii admin/create-theme
```

Потребоавлся пакет `npm`. Устновим его:

```
$ yum info npm
$ yum install npm
```

Для очистки кеша (может понадобится) используем:

```php
Yii::$app->cache->flush();
```
